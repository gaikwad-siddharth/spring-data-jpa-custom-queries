# 🔍 Spring Data JPA Custom Queries

A Spring Boot application demonstrating **custom query implementation** using **Spring Data JPA**, including **JPQL** and **Native SQL queries**.

---

## 📌 Project Overview

This project showcases how to go beyond basic CRUD operations by writing **custom database queries** using the `@Query` annotation in Spring Data JPA.

It helps in handling complex data retrieval scenarios commonly used in real-world applications.

---

## 🧠 Concepts Covered

- Spring Boot
- Spring Data JPA
- JPQL (Java Persistence Query Language)
- Native SQL Queries
- @Query Annotation
- Parameter Binding
- Custom Data Retrieval
